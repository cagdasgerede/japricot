<a name="valueOf(char[])"><!-- --></a><h3>
valueOf</h3>
<pre>public static <a href="../../java/lang/String.html" title="class in java.lang">String</a> <b>valueOf</b>(char[]&nbsp;data)</pre>
<dl>
<dd>Returns the string representation of the <code>char</code> array
 argument. The contents of the character array are copied; subsequent
 modification of the character array does not affect the newly
 created string.

<p>
</p></dd><dd><dl>
</dl>
</dd>
<dd><dl>
<dt><b>Parameters:</b></dt><dd><code>data</code> - a <code>char</code> array.
</dd><dt><b>Returns:</b></dt><dd>a newly allocated string representing the same sequence of
          characters contained in the character array argument.</dd></dl>
</dd>
</dl>
